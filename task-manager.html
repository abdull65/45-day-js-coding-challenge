Task Priority Manager - User's Guide

Adding a Task:

  1. Enter the task name.
  2. Enter the task priority (a number).
  3. Click "Add Task" to add and sort it by priority.

Editing a Task:

  1. Enter the task name to edit.
  2. Enter the new priority.
  3. Click "Edit Priority" to update and re-sort.

Deleting Task Priority:

  1. Enter the task name to delete.
  2. Click "Delete Task" to remove it.

Note:

  • Valid task names and unique names are essential.
  • The app sorts tasks by priority automatically.
